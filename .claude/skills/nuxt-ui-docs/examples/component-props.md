# Example 1: Component Props Query

## User Question

"What props does the Tooltip component accept?"

## Workflow

1. **Fetch index**: Access `/llms.txt` to locate Tooltip component documentation
2. **Identify page**: Find Tooltip component link in the component list
3. **Fetch specific page**: `https://ui.nuxt.com/components/tooltip`
4. **Use targeted prompt**: "List all props, slots, and usage examples for Tooltip"
5. **Provide answer**: Present comprehensive prop list with types and descriptions

## Expected Output

A complete listing of:
- All available props with TypeScript types
- Slot information if applicable
- Events the component emits
- Practical usage examples from the documentation
- Link to official documentation for reference

## Key Takeaway

Always fetch the specific component page with a targeted prompt rather than guessing API details from memory or older versions.
