# Example 3: Migration Assistance Query

## User Question

"I'm migrating from v2 to v3, what changed?"

## Workflow

1. **Fetch index**: Access `/llms.txt` to locate migration guide
2. **Identify page**: Find migration documentation link
3. **Fetch specific page**: `https://ui.nuxt.com/getting-started/migration/v2-v3`
4. **Use targeted prompt**: "List all breaking changes and migration steps"
5. **Provide answer**: Present step-by-step migration guide with code examples

## Expected Output

Comprehensive migration information including:
- List of breaking changes
- Step-by-step migration instructions
- Before/after code examples
- Component API changes
- Configuration updates required
- Deprecation warnings and alternatives
- Link to full migration guide

## Key Takeaway

Migration questions require comprehensive information. The migration guide page contains all breaking changes in one place, making it the primary source for version upgrade questions.
