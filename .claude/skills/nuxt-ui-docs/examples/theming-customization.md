# Example 2: Theming Customization Query

## User Question

"How do I customize the primary color?"

## Workflow

1. **Fetch index**: Access `/llms.txt` to identify theming documentation
2. **Identify page**: Locate theme configuration guide
3. **Fetch specific page**: `https://ui.nuxt.com/getting-started/theme`
4. **Use targeted prompt**: "Explain how to customize colors using CSS variables"
5. **Provide answer**: Show configuration example with color customization

## Expected Output

Configuration examples demonstrating:
- How to override CSS variables
- Where to place theme configuration (nuxt.config.ts or app.config.ts)
- Specific syntax for customizing primary, secondary, and other color tokens
- Preview of available color customization options
- Link to theming documentation

## Key Takeaway

Theming questions often span multiple documentation pages (theme, theming, customization). Prioritize the most relevant page based on the specific question.
